# Innovergence – Marvel Theme Hackathon Website

Deployment Link :-- https://innovergence.netlify.app



## Overview

This project was developed as part of the **Innovergence Hackathon** with a **Marvel-themed** concept. The objective was to design and build a modern, visually engaging, and production-grade website that reflects the energy of the Marvel universe while maintaining professional UI/UX standards suitable for a real-world tech product.

The website focuses on clean architecture, responsive design, and an intuitive user experience, making it suitable for both hackathon evaluation and future scalability.

---

## Key Highlights

* Marvel-inspired visual identity with a premium, modern look
* Industry-aligned UI/UX principles
* Fully responsive design (desktop, tablet, mobile)
* Clean code structure for maintainability
* Optimized performance and smooth navigation

---

## Features

* **Landing Page**: High-impact hero section with Marvel-style aesthetics
* **Event Information**: Hackathon details, theme, and objectives
* **Team / About Section**: Overview of contributors and vision
* **Dynamic UI Components**: Cards, sections, and animations for better engagement
* **Responsive Layout**: Works seamlessly across devices
* **Scalable Structure**: Easy to extend with backend or additional pages

---

## Tech Stack

* **Frontend**: HTML, CSS, JavaScript
* **Styling**: Modern CSS (Flexbox / Grid)
* **Design Approach**: Component-based UI, mobile-first design
* **Version Control**: Git

*(Tech stack can be extended to React, APIs, or backend services if required.)*

---

## Design Philosophy

The design combines **Marvel’s bold, cinematic feel** with **startup-grade product design**:

* Minimal clutter with strong visual hierarchy
* High contrast and sharp typography
* Developer-friendly layout that balances creativity and usability

## Installation & Setup

1. Clone the repository

   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory

   ```bash
   cd innovergence-marvel-hackathon
   ```
3. Open `index.html` in your browser

No additional dependencies are required for the basic setup.
